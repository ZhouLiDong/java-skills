# java-skills
对java项目有帮助的skill

1、read-jar：读取jar里面的类源码。
触发时机：
- 需要读取JAR文件中的类（第三方或内部依赖）
- 需要验证库类中是否存在某个方法或字段
- 需要理解框架类的实现逻辑
- 用户要求查看或检查JAR类的源代码
- 遇到编译错误，可能是由于API使用不当导致
- 在生成代码前需要确认方法的准确签名
- 需要了解某个框架组件的工作原理或调用链路时
限制：使用的cfr和quiltflower反编译工具版本支持的是java8，如果需要支持更早或者更新的java版本，建议替换这两个反编译工具为适合你项目的版本，也都是github开源的。
